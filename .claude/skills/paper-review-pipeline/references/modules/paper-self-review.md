# Module: Paper Self-Review (Feature-Parity Import)

This module preserves the distinctive workflow and checklist from the legacy `paper-self-review` skill.

## Core Features to Preserve

### 1) Structure Review

Check that each section is present and meets expectations:
- Abstract includes: problem, method, results, contributions.
- Introduction states: motivation/background + clear research objective.
- Method is detailed enough to be reproducible.
- Results support the conclusions.
- Discussion/Limitations covers limitations and future work (venue-dependent).

### 2) Logic Consistency Check

Verify coherence from questions -> methods -> experiments -> conclusions:
- Research questions match methodology.
- Experimental design supports hypotheses/claims.
- Interpretations are reasonable (no overclaim).
- Conclusions are evidence-backed.

### 3) Citation Completeness

- All citations in text appear in references.
- Reference format consistent.
- Key related works cited (positioning is credible).
- Citations actually support the attributed claim.

### 4) Figure/Table Quality

- Every figure/table has a clear title/caption.
- Supports the narrative and is referenced in text.
- Readable, correctly labeled, format matches venue norms.

### 5) Writing Clarity

- Language is concise and clear.
- Terminology used appropriately and consistently.
- Sentences and paragraphs are logically organized.

## Quality Checklist (Must Support)

Use this as a final "paper ready?" gate:

- [ ] Abstract includes problem, method, results, contributions
- [ ] Introduction clearly states research motivation
- [ ] Method is reproducible
- [ ] Results support conclusions
- [ ] Discussion/Limitations addresses limitations
- [ ] All figures/tables have captions and are referenced
- [ ] Citations are complete and accurate

## Review Process (Must Support)

1) Structure review
2) Content review (per section)
3) Citation check
4) Figure/table review
5) Writing quality review
6) Final checklist

## Best Practices (Optional but Preserved)

- Spaced review: wait 1-2 days before final self-review when possible.
- Multiple rounds: focus on one aspect per pass (structure vs citations vs prose).
- Print/read-aloud/reviewer-perspective techniques if the user wants them.
